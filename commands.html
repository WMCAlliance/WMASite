With a large collection of plugins comes a large collection of commands, and while we do try and use plugins that require the least amount of commands, sometimes it's just unavoidable. Below is a complete list of commands that you, as a player on our server, are able to perform. If you have permission issues with anything below, let me know.
The Basics
  • /spawn - Teleport to the local spawn
  • /motd - Read the main login message
  • /afk - Away From Keyboard mode toggle
  • /lobby - Teleport the the server lobby
  • /ping - Find out the exact amount of lag you have
 Public and Private Warps
  • /warps [PageNumber] - List of all warps you can access
  • /warp [WarpName] - Warp to a specified location
  • /setwarp [WarpName] - Create a public warp (max 5)
  • /setwarp -p [WarpName] - Create a private warp (max 20)
  • /mywarps [PageNumber] - List of your warps
Teleportation
  • /sethome - Set a private home warp
  • /home - Warp to home location
  • /tpr [Player] - Request teleport to go to another player
  • /tprh [Player] - Request teleport bring another player
  • /tpac - Accept teleportation request
  • /tpdeny - Deny teleportation request
  • /back - Teleport to previous location (eg. on death)
  • /backs - List of previous teleport locations (up to 5)
 Messaging and Player Lists
  • /list - Display list of ingame players, per world
  • /irclist - Display list of people in Internet Relay Chat
  • /online - See who's currently away from keyboard
  • /msg [Player] [Message] - Private message another player
  • /r [Message] - Quickly respond to private message
  • /ircmsg [User] [Message] - Private message a person in IRC
  • /ircreply [Message] - Quickly respond to private message
  • /me [Message] - Emote a message
Locking and Protection (tutorial)
  • //wand - Give yourself a WorldGuard tool (gold hoe)
  • //expand [Number] [Up/Down] - Expand selection size
  • //contract [Number] [Up/Down] - Contract selection size
  • /rg claim [Player] - Become owner of newly created region
  • /rg redefine [Region] - Update existing region
  • /rg addmember [Region] [Player] - Add members to a region
  • /rg removemember [Region] [Player] - Remove member
  • /lockette [Line] [Text] - Add members or timers to locks
 Hunger Games
  • /hg join 1 - Teleport and join Survival Games arena
  • /hg list 1 - List players that are playing/dead
  • /hg leave - Leave an arena you're currently in
  • /hg vote - Vote for a match to begin
  • /hg spectate 1 - Watch an existing match
  • /hg lq - Leave a game queue
Spleef, Capture The Flag and PVP
  • /warhub - Teleport to the arena portal lobby
  • /zone [spleef/ctf/pvp] - Teleport directly to an arena lobby
  • /join [Team Colour] - Quick-join/switch teams in arena
  • /teams - List teams in arena
  • /team [Message] - Private team chat
  • /leave - Leave an existing game
 Chess and Tetris
  • /chess invite [Player] - Invite player to Chess match
  • /chess join [Player] - Join invited Chess match
  • /chess [Yes/No] - Accept Chess match invitation
  • /tetris - Start/End Tetris match
  • /tspec [Player] - Spectate a Tetris match
  • /highscore - View Tetris high scores
Sign Commands + Editing
  • /signedit copy [Amount] - Copy a sign a number of times
  • /signedit [Line] [Text] - Text to put on a sign
  • /signedit cancel - Cancels the current edit
 Miscellaneous + Other
  • /lag - Display current world tickspeed (20 is normal)
  • /baninfo [Player] - Find out if a player is banned
  • /quit - Easy, immediate disconnect
